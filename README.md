# Cloud Storage Concept Simulation Python

This Python script demonstrates the fundamental concept of cloud storage by comparing a simulated local (on-premise) file system storage with a simulated cloud-based storage service. It highlights how cloud services abstract away infrastructure management, offering on-demand resources and scalability without direct hardware concerns.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal:
`python main.py`
